# California Department of Consumer Affairs (DCA) - Comprehensive URL Guide

**Agency Overview:** The Department of Consumer Affairs oversees 40+ professional licensing boards and bureaus that license millions of California professionals. Many boards use the BreEZe online system for applications, renewals, and license verification.

---

| Resource | URL | Description |
|----------|-----|-------------|

## BreEZe System Information

BreEZe is DCA's centralized online licensing system used by many (but not all) boards.

| Resource | URL | Description |
|----------|-----|-------------|
| BreEZe Portal | https://www.breeze.ca.gov/ | Main login portal for many boards |
| BreEZe Help | https://www.breeze.ca.gov/ | Login or create account |
| License Search | https://search.dca.ca.gov/ | Verify any DCA license |

## Common Business Pain Points

Based on research conducted for the BizAssessment project:

1. **Finding the right board** - With 40+ boards, businesses often struggle to identify which board licenses their profession

2. **Inconsistent processes** - Each board has different requirements, forms, and timelines

3. **BreEZe usability issues** - The centralized system has known UX challenges

4. **Opaque application status** - Limited visibility into where applications are in the review process

5. **Missed renewals** - Continuing education tracking and renewal deadlines cause license lapses

6. **Processing backlogs** - Some boards (BRN, CSLB) have 10-12+ week processing times
